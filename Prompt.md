# CampLog
Application that will provide tracking RV trips.  

- Multiple Users
- Users will be able to create multiple trips
- Trips will consist of multiple locations
- GPS coordinates can be associated with locations
- Trip: Name, Description, GPS, Start/End date, 
- Location: Name, Description, GPS, Start/End date



## Tech Stack
- .Net 10
- .Net Aspire for local development https://aspire.dev  Using CLI
- Razor Pages + HTMX + Pico CSS for frontend
- Keycloak for identity provider
- PostgreSQL for database
- Mobile friendly 
- Use the CSS styles below


## Use Cases
- Register as a new user
- Login/Logout
- Create/Edit/Update/Delete Trip
- Create/Edit/Update/Delete Locations for a trip
